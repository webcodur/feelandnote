# 셀럽 기본 정보 생성 룰북

셀럽의 기본 프로필 정보를 생성하는 가이드.

---

## JSON 출력 형식

```json
{
  "nickname": "셀럽 이름 (한국어로 작성, 예: 슬라보예 지젝)",
  "profession": "직군 코드",
  "title": "수식어 (2~8자, 대표작명/호칭/업적 중 하나)",
  "nationality": "국가 코드 (ISO 3166-1 alpha-2, 예: US, KR, GB, JP)",
  "gender": "성별 (남성: true, 여성: false, 불명: null)",
  "birth_date": "출생일 (YYYY-MM-DD 또는 -356 같은 기원전 연도)",
  "death_date": "사망일 (생존시 빈 문자열)",
  "bio": "인물 소개글 (2줄 분량, 한국어)",
  "quotes": "대표 명언 (한 문장)",
  "is_verified": false
}
```

## 수식어(title) 작성 가이드

- **용도**: 인물 이름 앞에 붙는 짧은 수식어
  - 예: "군주론의 마키아벨리", "철의 여인 마거릿 대처"
- **길이 제한**: 2~8자 이내로 최대한 짧게

### 우선순위 (위에서부터 적용)

1. **대표작**: 작품명 하나
   - 예: '군주론', '자본론', '코스모스'
2. **실제 호칭/별명**: 역사적 통용 호칭
   - 예: '철의 여인', '가왕', '독안룡'
3. **핵심 업적**: 가장 상징적인 것 하나
   - 예: '테슬라 창립자', '지동설'

### 금지

- 긴 설명형 문장
- 나열식 표현
- "~의 아버지/어머니" 같은 뻔한 표현

---

## 성별(gender) 필드 규칙

`gender` 필드는 **boolean** 타입이다.

| 값 | 의미 |
|-----|------|
| `true` | 남성 |
| `false` | 여성 |
| `null` | 알 수 없음 / 해당 없음 (논바이너리, 그룹, 불명) |

### 판단 기준

- **역사적 인물**: 문헌/기록 기반으로 명확히 판단
- **현대 인물**: 공식 프로필 또는 본인 발언 기반
- **불명확한 경우**: null로 설정

---

## 작성 규칙

1. **nickname**: 정확한 풀네임을 한국어로 작성
   - 예: 슬라보예 지젝, 알베르트 아인슈타인

2. **bio**: 100자 이내
   - 주어 없이 출신/직업을 짧게 서술
   - 마침표로 끊은 뒤 주요 업적 이어감

3. **nationality**: ISO 3166-1 alpha-2 국가 코드
   - 현대인이 아닌 경우 현재 해당 지역의 국가 코드로 재구성
   - 예: 알렉산더 대왕 → GR, 공자 → CN
   - 고대 국가로 특정 불가능한 경우 빈 문자열

4. **birth_date / death_date**
   - 정확한 날짜를 알 수 없으면 연도만 작성
   - 기원전은 음수로 표기 (예: -356)

5. **quotes**: 50자 이내
   - 알려진 것이 없으면 빈 문자열

6. **문자열 처리**
   - 큰따옴표는 작은따옴표로 대체

7. **is_verified**
   - **항상 false** (공식 인증은 수동 처리)
