# Feel&Note 개발 로드맵

## 개요

Phase별 점진적 개발. 각 Phase는 독립적으로 동작 가능한 단위.

---

## Phase 구성

```
Phase 1: MVP (핵심 루프) ✅ 완료
├── 인증 (Google, Kakao)
├── 콘텐츠 추가 (도서)
├── 기록 작성 (리뷰, 노트, 인용)
└── 기록관 조회

Phase 2: 기록 확장
├── 영화 카테고리 추가
├── 진행률 관리
└── 콘텐츠 상세 페이지

Phase 3: 소셜
├── 팔로우/친구
├── 피드
└── 좋아요/댓글

Phase 4: 게이미피케이션
├── 점수 시스템
├── 칭호 시스템
└── 놀이터 (티어리스트, 블라인드 게임)

Phase 5: 창작 & 확장
├── 창작 기능 (What If, 매체 전환, OST)
├── 카테고리 확장 (드라마, 애니, 게임)
└── 알림 시스템
```

---

## Phase 1: MVP ✅

### 목표
사용자가 도서를 검색하고, 기록관에 추가하고, 기록을 작성할 수 있다.

### 범위

| 기능 | 설명 | 상태 |
|------|------|------|
| 소셜 로그인 | Google, Kakao | ✅ |
| 도서 검색 | 알라딘 API 연동 | ✅ |
| 콘텐츠 추가 | 기록관에 도서 추가 | ✅ |
| 상태 관리 | 경험/관심 | ✅ |
| 기록 작성 | 리뷰/노트/인용 | ✅ |
| 기록관 조회 | 내 콘텐츠 목록 | ✅ |

### 데이터베이스

