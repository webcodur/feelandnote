# 01-01. 콘텐츠 매니저

## 개요

콘텐츠 매니저는 사용자의 문화 콘텐츠를 체계적으로 관리하는 기록관 시스템이다. 모든 기록(Review, Note, 창작)의 기반이 되는 핵심 모듈.

---

## 카테고리

### 지원 카테고리

| 카테고리 | 아이콘 | 범위 | 출시 단계 |
|----------|--------|------|-----------|
| 도서 | 📚 | 소설, 에세이, 자기계발, 만화, 웹소설 | Phase 1 |

## 콘텐츠 추가

### 추가 방식

1. **검색 추가**: 외부 DB에서 검색하여 추가
2. **직접 추가**: 메타데이터 수동 입력 (DB에 없는 콘텐츠)
3. **바코드/ISBN 스캔**: 도서 전용 (모바일)

### 검색 데이터 소스

| 카테고리 | 데이터 소스 |
|----------|-------------|
| 도서 | 알라딘 API, 국립중앙도서관 API |
| 영화/드라마 | TMDB, KMDB |
| 애니메이션 | AniList, MyAnimeList |
| 게임 | IGDB, Steam API |
| 공연 | 직접 입력 또는 제휴 DB |

### 추가 플로우

```
[+ 콘텐츠 추가] 클릭
    ↓
카테고리 선택
    ↓
검색어 입력
    ↓
검색 결과 목록
    ↓
콘텐츠 선택
    ↓
상태 선택 (경험/관심)
    ↓
기록관에 추가 완료
```

## 기록관 구조

### 뷰 모드

| 모드 | 설명 |
|------|------|
| 그리드 뷰 | 표지 썸네일 중심, 시각적 탐색 |
| 리스트 뷰 | 상세 정보 포함, 효율적 탐색 |

### 정렬 옵션

- 최근 추가순
- 제목순 (가나다)
- 별점순
- 진행률순
- 출시일순

### 필터 옵션

- 상태별 (감상중, 완료, 중단, 관심)
- 장르별
- 별점별
- 기간별 (추가일 기준)

## 진행률 관리

### 카테고리별 진행률 단위

| 카테고리 | 단위 | 예시 |
|----------|------|------|
| 도서 | 챕터/페이지 | 12/37 챕터 (32%) |
| 영화 | 완료 여부 | 완료/미완료 |
| 드라마 | 에피소드 | 8/16 에피소드 (50%) |
| 애니메이션 | 에피소드 | 12/24 에피소드 (50%) |
| 게임 | 직접 입력 (%) | 75% |
| 공연 | 완료 여부 | 완료/미완료 |

### 진행률 갱신

- Note의 구획 기록과 연동
- 구획 체크 시 자동 계산
- 수동 조정 가능

## 콘텐츠 상세 페이지

### 구성

```
┌─────────────────────────────────────────┐
│ ← 기록관으로                             │
├─────────────────────────────────────────┤
│ [표지]  제목                             │
│         저자/감독                        │
│         메타데이터                       │
│         태그                            │
│         진행률 바                        │
├─────────────────────────────────────────┤
│ [내 기록] [피드]                         │
├─────────────────────────────────────────┤
│ [Review] [Note] [창작]                  │
├─────────────────────────────────────────┤
│                                         │
│         (탭 내용)                        │
│                                         │
### Content

| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | 고유 식별자 |
| category | enum | 카테고리 |
| title | string | 제목 |
| creator | string | 저자/감독/개발사 |
| metadata | object | 카테고리별 메타데이터 |
| coverImage | string | 표지 이미지 URL |
| tags | string[] | 장르/태그 |
| externalId | string | 외부 DB 식별자 |

### UserContent

| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | 고유 식별자 |
| userId | string | 사용자 ID |
| contentId | string | 콘텐츠 ID |
| status | enum | 상태 (감상중/완료/중단/관심) |
| progress | number | 진행률 (0-100) |
| addedAt | datetime | 추가일시 |
| updatedAt | datetime | 수정일시 |

---

## 연동 모듈

- **Review**: 콘텐츠별 리뷰 연결
- **Note**: 콘텐츠별 노트 연결
- **창작**: 콘텐츠별 상상 연결
- **점수 시스템**: 콘텐츠 추가 시 활동 점수 +1
