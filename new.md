# 문화 경험 기록 플랫폼 설계

## 개요
독서, 영화, 게임, 공연, 미술 등 모든 문화 경험을 기록하고 공유하는 플랫폼.
독후감이 아닌, 개인이 기억을 잃지 않고 감상을 흥미롭게 이어가는 도구.

## 핵심 구조

### 2-Part 시스템
- **Part 1: 기록 관리** (기본 비공개, 선택적 공개 가능)
- **Part 2: 리뷰** (기본 공개, 선택적 비공개 가능)

---

## POINT 시스템

### 개념
**인상적인 문구/구절을 저장하고, 나중에 메모에서 참조할 수 있는 북마크 시스템**

### 작동 방식
1. **저장**: 감상 중 기억하고 싶은 문구를 발견하면 POINT로 저장
2. **참조**: 메모/리뷰 작성 시 저장한 문구를 불러와서 사용
3. **연결**: 한 문구를 여러 메모에서 참조 가능, 역추적도 가능

### 저장 내용
- **text** (필수): 저장할 문구 전체
  - 예: `"인생은 초콜릿 상자와 같다"`
- **label** (선택): 짧은 별명 (긴 문구를 짧게 부를 때)
  - 예: `"초콜릿 명언"`
- **location** (선택): 출처 위치
  - 예: `"p.142"`, `"01:23:45"`, `"3장"`

### 사용 예시
```
[저장]
text: "사람은 누구나 자기 인생의 주인공이다"
label: "주인공 대사"
location: "p.87"

[메모 작성 시]
"[[주인공 대사]]가 5장 반전과 연결된다"
또는
"[[사람은 누구나 자기 인생의 주인공이다]]는 말이 울림이 컸다"

[클릭하면]
→ 원문: "사람은 누구나 자기 인생의 주인공이다" (p.87)
→ 이 문구를 참조한 다른 메모들
```

### 기능
- 저장한 문구 목록 조회 (책별/전체)
- 메모 작성 시 드롭다운에서 선택 삽입
- 참조된 문구 클릭 시 원문 + 출처 표시
- 한 문구를 참조한 모든 메모 조회
- 자주 참조하는 문구 통계

---

## Part 1: 기록 관리 (기본 비공개)

감상 진행 중 실시간으로 작성하는 개인 메모

### 순간 포착
- 강렬한 장면/문장 즉시 저장
- 그 순간 든 생각/질문
- 감정 변화 체크포인트

### 진행 노트
- 챕터/회차별 메모
- 복선/의문점 기록
- 예측/기대 적기

### 개인 반응
- 공감/거부감 지점
- 내 경험 떠오른 순간
- 멈춰서 생각한 부분

**특징**: POINT 시스템을 활용하여 모든 메모에 저장된 문구 참조 가능

---

## Part 2: 리뷰 (기본 공개)

감상 완료 후 전체를 돌아보며 작성하는 정리된 감상

### 1. 경험 스냅샷
- 언제 어디서 어떤 상황에서 감상했는가 (시간/장소/동행/계기)
- 전체 감정 곡선 (시작-중반-끝)
- 이 작품 전후로 달라진 것

### 2. 핵심 포착
- 3줄 요약
- 작품의 질문 vs 내 질문
- 가장 강렬했던 순간
- 선택한 인용구들

### 3. 재창작 playground
- IF 시나리오 (다른 결말/선택지 상상)
- 가상 캐스팅 (다른 배우/감독/작가라면)
- 외전/프리퀄/속편 아이디어
- 다른 매체로 전환한다면 (소설→영화, 게임→드라마 등)
- OST 교체/삽입 상상
- 캐릭터 시점 바꾸기
- 시대/배경 변경 버전

---

## 플랫폼 제공 기능 vs 유저 작성 콘텐츠

### 플랫폼 제공
- 작품 메타데이터 (제목, 창작자, 발표연도, 매체 정보)
- 실용 정보 (구매/대여/스트리밍 링크, 절판 여부)
- 작품 관계망 자동 추천 (유사 작품, 관련 작품)
- 통계 및 시각화 (취향 분석, 소비 패턴)

### 유저 작성
- 개인의 경험과 해석
- 감정과 생각의 기록
- 창작적 상상과 놀이

---

## 차별화 포인트

1. **POINT 시스템**: 인상적인 문구를 저장하고 메모에서 참조
2. **진행 중/완료 후 분리**: 실시간 메모와 정리된 리뷰 구분
3. **재창작 playground**: 단순 평가를 넘어선 창작적 참여
4. **선택적 공개/비공개**: 개인 기록과 공유 콘텐츠의 유연한 관리
5. **범용 문화 경험**: 모든 매체를 하나의 시스템으로 통합
