# 04-01. 창작

## 개요

창작은 작품 기반 창작과 상상의 공간이다. What If 시나리오, 매체 전환 투표, OST 상상을 통해 단순 소비자가 아닌 창작자로서 작품과 관계 맺는 경험을 제공한다. 전체 공개 고정.

---

## 핵심 기능

### 1. What If

"만약 ~했다면?" 상상의 시나리오를 작성하고 공유한다.

#### What If 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| 다른 결말 | 원작과 다른 엔딩 | "해리가 죽었다면?" |
| 다른 선택 | 캐릭터의 다른 선택 | "해리가 슬리데린을 선택했다면?" |
| 시대 변경 | 다른 시대 배경 | "조선시대 호그와트?" |
| 배경 변경 | 다른 장소/세계관 | "현대 서울 배경이라면?" |
| 시점 전환 | 다른 캐릭터 시점 | "드레이코 시점에서 본 이야기" |
| 프리퀄 | 본편 이전 이야기 | "호그와트 창립 이야기" |
| 외전 | 사이드 스토리 | "네빌의 숨겨진 이야기" |
| 속편 | 본편 이후 이야기 | "해리의 자녀 세대" |

#### 작성 요소

| 요소 | 필수 | 설명 |
|------|------|------|
| 유형 선택 | O | 위 8가지 중 1개 선택 |
| 본문 | O | 상상 시나리오 서술 (최대 10,000자) |

---

### 2. 매체 전환

"이 작품이 다른 매체로 만들어진다면?" 투표와 제안을 통해 매체 전환을 상상한다.

#### 매체 옵션

| 매체 | 아이콘 |
|------|--------|
| 영화 | 🎬 |
| 드라마 | 📺 |
| 연극 | 🎭 |
| 웹툰 | 📚 |
| 오디오북 | 🎧 |
| 게임 | 🎮 |

#### 작성 요소

| 요소 | 필수 | 설명 |
|------|------|------|
| 매체 선택 | O | 위 6가지 중 1개 선택 |
| 캐스팅 제안 | X | 역할 - 배우/성우 매핑 (복수 가능) |
| 제작 방향성 | X | 제작 방향에 대한 의견 서술 |

#### 투표 시스템

- "이 작품, 꼭 만들어주세요!" 투표
- 목표 인원 설정 (예: 1,000명)
- 목표 달성 시 참여자 전원 알림 + 보너스 점수

---

### 3. OST 상상

"이 작품에 음악을 입힌다면?" 장면별 선곡과 음악적 상상을 공유한다.

#### 작성 요소

| 요소 | 필수 | 설명 |
|------|------|------|
| 장면별 OST | X | 장면/챕터 - 곡 정보 - 선곡 이유 (복수 가능) |
| 전체 방향성 | X | 전체 OST 방향성 코멘트 |

#### 장면별 OST 구조

| 필드 | 설명 |
|------|------|
| 장면/챕터 | 어떤 장면에 적용할지 |
| 곡 정보 | 곡 제목 - 아티스트 |
| 선곡 이유 | 왜 이 곡을 선택했는지 |

---

## 공개 설정

**전체 공개 고정**

창작은 상상을 공유하고 소통하는 공간이므로 모든 작성물이 전체 공개.

---

## 작성 규칙

### 작성 조건
- 콘텐츠가 기록관에 추가되어 있어야 함
- 상태 무관

### 복수 작성
- 콘텐츠당 **복수의 창작물 작성 가능**
- What If 여러 개, 매체 전환 여러 개 등

### 수정/삭제
- 언제든 수정 가능
- 삭제 시 완전 삭제

---

## 상호작용

### 좋아요
- 다른 유저의 창작물에 좋아요 가능
- 좋아요 취소 가능

### 댓글
- 다른 유저의 창작물에 댓글 작성 가능
- 댓글에 대한 답글 지원 (1depth)

### 매체 전환 투표
- 특정 매체 전환 제안에 투표 참여
- 투표 취소 가능
- 투표 현황 실시간 표시

## 데이터 모델 (개념)

### Creation (공통)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | 고유 식별자 |
| userId | string | 작성자 ID |
| contentId | string | 콘텐츠 ID |
| type | enum | 유형 (whatif, media, ost) |
| likeCount | number | 좋아요 수 |
| commentCount | number | 댓글 수 |
| createdAt | datetime | 작성일시 |
| updatedAt | datetime | 수정일시 |

### WhatIf (What If 전용)

| 필드 | 타입 | 설명 |
|------|------|------|
| creationId | string | 창작 ID |
| subType | enum | 유형 (다른결말, 다른선택, ...) |
| body | string | 본문 |

### MediaConversion (매체 전환 전용)

| 필드 | 타입 | 설명 |
|------|------|------|
| creationId | string | 창작 ID |
| targetMedia | enum | 목표 매체 |
| casting | array | 캐스팅 배열 [{role, actor}] |
| direction | string | 제작 방향성 |
| voteCount | number | 투표 수 |
| voteGoal | number | 목표 투표 수 |

### OstImagination (OST 상상 전용)

| 필드 | 타입 | 설명 |
|------|------|------|
| creationId | string | 창작 ID |
| tracks | array | 트랙 배열 [{scene, song, reason}] |
| overallDirection | string | 전체 방향성 |
