## 1. 티어리스트

### 개요
S/A/B/C/D 5단계 등급으로 내가 경험한 작품들을 랭킹하는 기능.

### 등급 체계

| 등급 | 의미 | 색상 |
|------|------|------|
| S | 최고, 인생작 | 금색 |
| A | 매우 좋음 | 보라 |
| B | 좋음 | 파랑 |
| C | 보통 | 초록 |
| D | 별로 | 회색 |

### 작성 방식
- 드래그 앤 드롭으로 작품을 등급에 배치
- 같은 등급 내 순서 조정 가능
- 기록관의 "경험" 상태 작품만 대상

### 리스트 종류

| 유형 | 설명 |
|------|------|
| 전체 | 모든 카테고리 통합 |
| 카테고리별 | 도서/영화/드라마 등 개별 |
| 장르별 | 판타지/SF/로맨스 등 |
| 연도별 | 2024년 베스트, 2023년 베스트 등 |
| 커스텀 | 사용자 정의 주제 (예: "울었던 영화들") |

### 공유 기능
- 이미지로 저장 (PNG)
- SNS 공유 (트위터, 인스타그램)
- 링크 공유

### UI 구조

```

│                                         │
│ [전체▼] [도서] [영화] [+ 새 리스트]     │
├─────────────────────────────────────────┤
│                                         │
│ S │ [표지][표지][표지]                  │
│ ──┼───────────────────────────────────  │
│ A │ [표지][표지][표지][표지]            │
│ ──┼───────────────────────────────────  │
│ B │ [표지][표지]                        │
│ ──┼───────────────────────────────────  │
│ C │ [표지][표지][표지]                  │
│ ──┼───────────────────────────────────  │
│ D │ [표지]                              │
│                                         │
├─────────────────────────────────────────┤
│ 미배치 작품: 12개                       │
│ [표지][표지][표지]...                   │
├─────────────────────────────────────────┤

└─────────────────────────────────────────┘
```

## 2. 블라인드 게임

### 개요
제목/썸네일 없이 감상평만 보고 작품을 맞추는 퀴즈 게임.

### 게임 방식

1. 랜덤 Review가 제목/표지 없이 표시
2. 사용자가 작품명 입력
3. 정답/오답 판정
4. 힌트 순차 공개 가능

### 힌트 시스템

| 순서 | 힌트 | 공개 시 감점 |
|------|------|-------------|
| 1 | 카테고리 (도서/영화 등) | -1점 |
| 2 | 출시 연도 | -1점 |
| 3 | 장르 | -1점 |
| 4 | 저자/감독 | -2점 |

### 점수 체계

| 조건 | 점수 |
|------|------|
| 힌트 없이 정답 | +5점 |
| 힌트 사용 후 정답 | +5 - 감점 |
| 오답 | 0점 |
| 연속 정답 보너스 | 스트릭 × 1 |

### 스트릭 시스템
- 연속 정답 시 스트릭 증가
- 오답 시 스트릭 리셋
- 스트릭 보너스: 현재 스트릭 수만큼 추가 점수

### 리더보드
- 일간/주간/월간/전체 랭킹
- 내 순위 표시

### UI 구조
