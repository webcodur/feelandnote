## 작업 절차

### 유형 A (자기등장 감사)
1. 해당 셀럽의 user_contents + contents 전수 조회
2. 각 항목 판별: "이 인물이 실제로 감상한 것인가?"
3. 자기등장/자기소재 항목 삭제
4. 부족하면 opus 에이전트로 재수집
5. 영어 제목 → 한국어 정식 출판명 매칭

### 유형 B (한국어 출판명 매칭)
1. 영어 제목 목록 추출
2. 한국어 정식 출판명 매칭 (Naver API 검증)
3. contents 테이블 UPDATE (title, creator, thumbnail_url)
