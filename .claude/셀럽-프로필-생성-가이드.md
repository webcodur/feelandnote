# 셀럽 프로필 생성 가이드

셀럽 계정을 생성하고 관리하는 에이전트 사용법.

---

## 에이전트 목록

| 에이전트 | 용도 | 호출 예시 |
|----------|------|----------|
| `celeb-account-creator` | 전체 처리 (기본정보 + 영향력 + 감상철학) | "스티브 잡스 셀럽 계정 만들어줘" |
| `celeb-basic-profile` | 기본 정보만 | "스티브 잡스 기본 정보 생성해줘" |
| `celeb-influence` | 영향력 평가만 | "스티브 잡스 영향력 평가해줘" |
| `celeb-philosophy` | 감상 철학만 | "스티브 잡스 감상 철학 작성해줘" |

---

## 전체 처리 (한번에)

### 기본 사용

```
윌리엄 셰익스피어
```
→ 기본 정보 + 감상 철학 생성

### 컨텐츠 수집까지

```
알베르트 아인슈타인 계정 만들고 컨텐츠 수집까지 해줘
```
→ 기본 정보 + 감상 철학 + 컨텐츠 수집

---

## 개별 처리

### 기본 정보만

```
스티브 잡스 기본 정보 생성해줘
```

생성 항목:
- 이름, 직군, 수식어
- 국적, 성별, 생몰일
- 소개, 명언

### 영향력만

```
알베르트 아인슈타인 영향력 평가해줘
```

평가 항목:
- 6개 영역 (정치, 전략, 기술, 사회, 경제, 문화) - 각 0~10점
- 통시성 - 0~40점
- 총점 100점 / S~D 등급

### 감상 철학만

```
빌 게이츠 감상 철학 작성해줘
```

작성 내용:
- 결정적 콘텐츠 경험
- 감상 방식의 특이점
- 직업/업적과의 연결
- 4문단 구조

---

## 팁

### 정확한 결과를 위해

- 풀네임을 사용하면 더 정확 (예: "일론 머스크" 대신 "일론 리브 머스크")
- 동명이인이 있는 경우 특징을 추가 (예: "테슬라 창업자 일론 머스크")

### 수정/업데이트

- 기존 셀럽 정보 수정: "스티브 잡스 영향력 다시 평가해줘"
- 특정 항목만 수정: "스티브 잡스 수식어만 바꿔줘"

### DB 저장

- 생성 후 자동으로 Supabase에 저장됨
- 저장 실패 시 오류 메시지와 해결 방안 제시

## 출력 예시

### 전체 처리

```
**[작업 요약]** 셀럽 계정 생성: 스티브 잡스 (Steve Jobs)
- 기본 정보: ✅
- 감상 철학: ✅
- 컨텐츠 수집: ⏭️ (건너뜀)
---
(상세 내용)
```

### 영향력 평가

```
**[작업 요약]** 영향력 평가: 알베르트 아인슈타인
- 총점: 74/100
- 등급: A
---
(JSON 출력)
```
